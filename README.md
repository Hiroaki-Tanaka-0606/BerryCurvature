# BerryCurvature
Calculate energy eigenvalue and Berry curvature of a matrix Hamiltonian

## Usage
```bash
$ eigen.o INPUT_FILE
$ berry.o INPUT_FILE
```
The default name of the input file (used when missing the argument ```INPUT_FILE```) is **input.dat** (defined in **initialize.cpp**).
